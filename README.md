# ACM Programming Contest Solutions

C++ solutions and problem assets from ACM-style programming contest practice, circa 2011.

## What this shows

- Competitive programming problem solving in C and C++.
- Small, focused implementations with sample inputs and contest PDFs.
- Algorithm practice across parsing, simulation, sorting, and data-structure exercises.

## Archive Status

This repository is preserved as a public reference for contest-style problem solving. It is not maintained as a single active build target, but the source files are intentionally left readable and close to their original form.
